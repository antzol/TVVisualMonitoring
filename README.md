# TVVisualMonitoring
Application package for visual monitoring of television programs (IPTV, DVB-T2 and etc.).

**The project is under development.**

The project currently consists of
- tvvm-server is an application for receiving transport streams and displaying TV programs in the form of a mosaic;
- tvvm-configurator is an application for configuring this application package.
